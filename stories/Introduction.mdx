<div className='subheading'>Get Started</div>

- Install <strong>npm-cli-logi</strong>:

<Source dark language='bash' code='yarn global add npm-cli-login' />

- Login no git:

<Source
  dark
  language='bash'
  code='yarn login --registry=https://npm.pkg.github.com'
/>

- Instalar:

<Source dark language='bash' code='yarn add @j-meira/mui-theme' />

- Use MultiProvider:

<Source
  dark
  language='typescript'
  code={`
import { MultiProvider, Toast } from '@j-meira/mui-theme';
import { Loading } from './components';

import { AppRoutes } from './routers';

export const App = () => (
  <MultiProvider
    adapterLocalePtBR
    palette={{
      primary: {
        light: '#6fddff',
        main: '#29abe2',
        dark: '#007cb0',
        contrastText: '#fff',
      },
      secondary: {
        light: '#74706f',
        main: '#494544',
        dark: '#221e1d',
        contrastText: '#fff',
      },
    }}
  >
    <AppRoutes />
    <Loading />
    <Toast />
  </MultiProvider>
);

`}
/>

- Import style file:

<Source
  dark
  language='CSS'
  code={`@import '~@j-meira/mui-theme/dist/scss/index.scss';`}
/>

- Use the components:

<Source
  dark
  language='typescript'
  code={`
  import { Button } from '@j-meira/mui-theme';

export const AppRoutes = () => (
  <Button>Basic</Button>
);
  `}
/>
